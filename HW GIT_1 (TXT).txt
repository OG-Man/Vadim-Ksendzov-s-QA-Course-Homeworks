
TXT
 1. Создать внешний репозиторий c названием TXT.
    Done

 2. Клонировать репозиторий TXT на локальный компьютер.
    git clone {SSH key} 
   
 3. Внутри локального TXT создать файл “new.txt”.
    touch TXT/new.txt

 4. Добавить файл под гит.
    cd TXT/
    git add new.txt

 5. Закоммитить файл.
    git commit -m 'Add new empty .txt file'

 6. Отправить файл на внешний GitHub репозиторий.
    git push

 7. Отредактировать содержание файла “new.txt” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате TXT.

nano new.txt  -> Ввести текст

First name: Oleg.
Last name: Golovin.
Age: 34.
Pets counter: 2.
Salary at wish: To Infinity and Beyond! (c)

Сохранить и выйти

 8. Отправить изменения на внешний репозиторий.
    git add new.txt
    git commit -m 'Add some info about me'
    git push

 9. Создать файл preferences.txt
    touch preferences.txt

 10. В файл preferences.txt” добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате TXT.

nano preferences.txt

  Favorite movie: Forrest Gump.
  Favorite TVseries: Stranger Things, Futurama, Monty Python’s Flying Circus, Twin Peaks, The Office (US), Firefly, Gravity Falls.
  Favorite dish: potato-based gratin.
  Favorite season: summer.
  Countries I want to visit: Ireland, Japan, New Zealand, Montenegro.

 11. Создать файл skills.txt добавить информацию о скиллах которые будут изучены на курсе в формате TXT

    touch skills.txt
    nano skills.txt
 
manual testing
writing test cases
writing check lists
writing bug reports
GIT fundamentals
GitBash commands
mobile testing
API testing using Postman
SQL fundamentals

 12. Сделать коммит в одну строку.
    git add -A && git commit -m 'Add 2 new .txt files'

 13. Отправить сразу 2 файла на внешний репозиторий.
    git push
